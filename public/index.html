Documentacion API - Elias Escalante - Entrega Final

Obtener el listado de todos los productos en la base de datos

/products

Para agregar un producto en tiempo real o eliminarlo de la base de datos

/realtimeproducts

Crear un carrito

POST /api/carts

{
    "products": []
}

Obtener el producto del carrito

GET /api/carts/:cid

Agregar un producto al carrito

POST /api/carts/:cid/product/:pid

{
    "quantity": 1
}
